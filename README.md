# StoneSight AI | Premium Stone Visualization

StoneSight AI is a high-fidelity architectural visualization tool that transforms kitchen and interior spaces with premium stone collections. Powered by NVIDIA NIM and Cosmos 3, it delivers photorealistic image editing and cinematic 4K walkthroughs.

View your app in AI Studio: https://ai.studio/apps/06f2d2c0-1b21-45fc-9cf6-78b6b6b19254

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `NVIDIA_API_KEY` in [.env](.env) to your NVIDIA API key
3. Run the app:
   `npm run dev`
